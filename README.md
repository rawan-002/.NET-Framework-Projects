# .NET Framework Projects 

Welcome to my repository for .NET Framework development! This repository serves as a portfolio of various desktop applications built using **C#** and **Windows Forms (WinForms)**.

## Overview
The goal of this repository is to document my journey in mastering C# and desktop application development. Each project focuses on core programming concepts, including:
* **GUI Design:** Creating intuitive and user-friendly interfaces.
* **Event-Driven Programming:** Managing user interactions effectively.
* **Business Logic:** Implementing calculations and data management within the code.

A comprehensive application for a pizza restaurant that allows users to customize their orders.
* **Key Features:**
    * **Dynamic Pricing:** Calculates total cost based on size, toppings, and crust type using the `Tag` property.
    * **Live Summary:** Updates the "Order Summary" section instantly as the user makes choices.

* **Path:** [./Pizza-Project](https://github.com/rawan-002/.NET-Framework-Projects/tree/main/Pizza-Project)
##  Requirements
To run these projects locally, you will need:
1.  **Visual Studio** (2019 or later recommended).
2.  **.NET Framework 4.7.2** (or the specific version mentioned in the project folder).

## Getting Started
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/rawan-002/.NET-Framework-Projects.git](https://github.com/rawan-002/.NET-Framework-Projects.git)
    ```
2.  Navigate to the specific project folder.
3.  Open the `.sln` (Solution) file in Visual Studio.
4.  Press `F5` or click **Start** to run the application.
